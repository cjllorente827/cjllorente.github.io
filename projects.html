Ndustria, a Python Library for Hi-performance Data Mining and Analysis

Computational research has a problem. As data collection methods improve and simulations increase their resolution, the amount raw data available for analysis skyrockets. But if the software built to analyze this data doesn't improve along with it, then a new bottleneck is created at the analysis level. Ndustria works to alleviate this bottleneck with as few code changes as possible, by transforming user-defined Python code into an error-resistant, parallelized analysis pipeline.
